   Загальні відомості
   General Information

   Модуль "Ремонт" призначений для відображення інформації про ремонтні роботи, які проводяться у місті/районі/області. На основі даних із завантажених в систему файлів будується інтерактивна карта ремонтів.
   Module "Repair" is intended to display information about repairs that are held in the city / district / region. Based on data from the uploaded files builded interactive map of repairs.

   Довідник робіт.
   Reference works.

   Довідник робіт містить категорії та підкатегорії (види) робіт.
   Reference works containing categories and subcategories (species) work.

   Кожен файл з ремонтними роботами відповідає певній категорії робіт. Стовбець файлу "Робота" - відповідає підкатегорії (виду) робіт за довідником.
   Each file with repair work meets certain categories of works. Column of file "Work" - is responsible for sub-category (type) of works for reference.

   Слої
   пласти
   Layers

   В модулі Слої можна завантажити файли: 1 файл = 1 категорія робіт.
   In the module Layers you can upload files: 1 file = 1 category of work.

   Файл із даними про ремонтні роботи повинен містити наступні стовбці.
   Data file repair work should include the following columns.

   Балансоутримувач
   Balanceholder
   Адреса
   Address
   Адреса 1 (кінцева адреса: вказується за потреби, якщо це, наприклад, ремонт дороги)
   Address 1 (final address: indicated as needed, for example, if this repairing roads)

   Об'єкт (конкретний об'єкт, на якому проводяться роботи)
   Facility (concrete object that is being made)
   Робота (вид робіт з Довідника робіт )
   Work (type of work from Reference works)
   Вартість (в грн.)
   Price (in USD).
   Дата ремонту
   Date of repair
   Гарантія (дата придатності проведених робіт)
   Warranty (date suitability of performed works)
   Додаткова інформація
   Additional Information


   Адреси можна вказувати як точну геопозицію (наприклад, "49.23363, 28.46146"), так і назву (наприклад, "площа Гагаріна 4/1").
   Addresses can be specified as an exact geolocation (eg "49.23363, 28.46146") and name (eg "Gagarin Square 4/1").
